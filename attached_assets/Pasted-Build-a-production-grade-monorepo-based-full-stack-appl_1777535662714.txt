Build a production-grade monorepo-based full-stack application called "Eden AI Workspace", inspired by eden.so.
 
This is NOT a demo project. It must follow real-world scalable architecture using a monorepo setup.
 
---
 
🏗️ MONOREPO SETUP (MANDATORY)
 
Use Turborepo + PNPM workspaces
 
Root structure:
 
/apps
/web        → Next.js frontend (App Router)
/api        → Backend (Next.js API routes or Express)
 
/packages
/ui         → Shared UI components (ShadCN आधारित reusable components)
/db         → Prisma schema + client
/ai         → NVIDIA NIM integration + RAG logic
/config     → Shared configs (tsconfig, eslint, tailwind)
/types      → Shared TypeScript types
 
---
 
⚙️ TECH STACK
 
Frontend (apps/web):
 
- Next.js (App Router)
- TypeScript
- TailwindCSS
- ShadCN UI
- Zustand
 
Backend (apps/api):
 
- Node.js / Next.js API routes
- Prisma ORM
- PostgreSQL (Neon/Supabase)
 
AI Layer (packages/ai):
 
- NVIDIA NIM API (PRIMARY)
- Free hosted models only
- OpenAI-compatible fallback support
 
Storage:
 
- Cloudinary or S3
 
Realtime:
 
- Socket.IO
 
---
 
🧠 AI + RAG SYSTEM (IN packages/ai)
 
Implement modular services:
 
- ingestionService.ts
- processingService.ts
- embeddingService.ts
- retrievalService.ts
- generationService.ts
 
Pipeline:
 
1. Ingestion → files + URLs
2. Processing → extract + clean + chunk
3. Embeddings → NVIDIA NIM
4. Storage → PostgreSQL (pgvector)
5. Retrieval → semantic search
6. Generation → LLM response (streaming)
